Thanks for purchase UGUI KIT FLAT.

 Get Started:---------------------------------------------------------------------
- Import the package into the unity project.
- Watch sample scenes to know that it each prefabricated.
- Drop and start design you UI interface!.

 Usage:--------------------------------------------------------------------------

- Select a prefab from 'prefabs' folder of package.
- Drag and drop this into the UI Canvas of your scene.
- Modify this to your taste and ready.

 Use Progress Bar:--------------------------------------------------------------

- For use a progress bar and control the value of this by scripting.
    - create a reference variable of bl_ProgressBar in your script, eg: public bl_ProgressBar ProgressBarHealth;
    - in loop funtion or where you need modify the value of bar, call like this:
    ProgressBarHealth.Value = MyFloatValue;//where MyFloatValue is the value of your script as reference for the bar.
